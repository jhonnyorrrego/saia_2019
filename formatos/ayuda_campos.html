FORMATO DE LOS PARÁMETROS

Arboles
  • Ruta del xml (relativa desde la carpeta del formato)
  • Tipo de control (1=> checkbox 2=>radiobutton)
  • Modo calcular numero de nodos hijo //no se está usando
  • Forma de carga (0=>autoloading 1=>smartXML)
  • Busqueda (0=>No 1=>Si)
  • Almacenar (0=>iddato 1=>valordato)
  • Tipo de arbol (0=>funcionarios 1=>series 2=>dependencias 3=>Otro (se debe sacar el dato) 4=>Sale de la tabla enviada a test_serie.php?tabla=nombre_tabla 5=>rol)
Ej: Arbol de funcionarios(rol) Arbol de dependencias

Boton de Selección, Cuadro de Chequeo, Lista Deplegable

  • Estático: Lista de valores (guardar,mostrar;guardar2,mostrar2...) 0,No;1,Si
  • Dinámico: Consulta sql (Select col1 as id,col2 as nombre from tabla1 order by nombre) se guarda el id y se muestra el nombre
Select Dependiente

  • nombre del select padre; sql select padre| nombre del select hijo; sql select hijo.... (ej: Departamento;select iddepartamento as id,nombre from departamento order by nombre| Municipio; select idmunicipio as id,nombre from municipio where departamento_iddepartamento=)
Area de Texto

  • Tipo de componente: avanzado, basico, formatos, noeditor, buscador
Remitente

  • Tipo de componente: unico o mutiple
  • Campos con autocompletar: nombre,identificacion
  • Campos normales: identificacion,cargo,empresa,direccion,telefono,email,titulo,ciudad
  • Función que se va a usar para mostrar los datos en pantalla: no es obligatorio, si se llena la función debe existir en el archivo formatos/librerias/funciones_ejecutor.php
Si se deja vacio los valores por defecto son los usados por la carta: multiple@nombre,identificacion@cargo,empresa,direccion,telefono,email,titulo,ciudad

Detalle, Item

  • Id del formato padre
Archivo

  • Extensiones aceptadas: extensiones separadas por pipe "|" Ej: xls|doc|pdf|jpg
  • Tipoo: "unico" o "multiple" Ej: xls|doc|pdf|jpg@unico
    • Si no se llena se toman las establecidas en la configuración
Lista Numérica

  • Valor inicial: valor mínimo permitido
  • Valor final: valor máximo permitido
  • Incremento: incremento entre cada opcion
  • Bloquear entrada por teclado: 1 o 0
Ej: 0@100@5@0
